A. Neural Network Fundamentals

Core Concepts
  • Perceptron & Neurons ‚Äî Weighted sum + bias ‚Üí activation function. Foundation of all neural networks.
  • Activation Functions ‚Äî ReLU, LeakyReLU, Sigmoid, Tanh, GELU, Swish. Explain saturation vs sparsity tradeoffs.
  • Loss Functions ‚Äî MSE, Cross-Entropy, Binary Cross-Entropy, CTC, Triplet Loss, Focal Loss.
  • Backpropagation ‚Äî Chain rule for gradient computation. Weight update: w ‚Üê w - Œ∑‚àÇL/‚àÇw.
  • Optimizers ‚Äî SGD, Momentum, RMSprop, Adam, AdamW. Know weight decay vs L2 regularization.
  • Initialization ‚Äî Xavier/Glorot, He initialization. Prevents vanishing/exploding gradients.

B. Architecture Types

Model Design
  • Feedforward (MLP) ‚Äî Dense layers for tabular and structured data tasks.
  • Convolutional (CNN) ‚Äî Conv layers, pooling, stride. ResNet, VGG, EfficientNet for vision.
  • Recurrent (RNN/LSTM/GRU) ‚Äî Sequential data, hidden states. Note vanishing gradient issues.
  • Transformers ‚Äî Self-attention, multi-head attention, positional encoding. Dominant in NLP & vision (ViT).
  • Autoencoders (VAE) ‚Äî Representation learning, denoising, dimensionality reduction, anomaly detection.
  • GANs ‚Äî Generator + discriminator adversarial training for generative modeling.
  • Graph Neural Networks ‚Äî Message passing & aggregation on graphs (GCN, GAT, GraphSAGE).

C. Training Techniques

Stability & Performance
  • Regularization ‚Äî L1/L2, weight decay, early stopping, label smoothing.
  • Normalization ‚Äî Batch Norm, Layer Norm, Group Norm, Instance Norm. Stabilizes training.
  • Dropout ‚Äî Random neuron deactivation during training. Prevents overfitting.
  • Data Augmentation ‚Äî Flips, crops, rotations, color jitter, mixup, cutout, SpecAugment (audio).
  • Learning Rate Schedules ‚Äî Step decay, cosine annealing, warmup, ReduceLROnPlateau, cyclical LR.
  • Mixed Precision Training ‚Äî FP16/BF16 for faster training. Gradient accumulation for large batches.
  • Transfer Learning ‚Äî Fine-tune pretrained models. Linear probing, feature extraction.

D. Tasks & Applications

Real-world Use
  • Computer Vision ‚Äî Classification, object detection (YOLO, Faster R-CNN), semantic/instance segmentation (U-Net, Mask R-CNN).
  • Natural Language Processing ‚Äî Language modeling (GPT, BERT), translation, QA, summarization, sentiment analysis.
  • Speech & Audio ‚Äî ASR (Whisper), TTS, speaker recognition, audio classification, music generation.
  • Recommender Systems ‚Äî Collaborative filtering, content-based, embeddings, ranking, two-tower models.
  • Generative AI ‚Äî Image/video generation, diffusion models (Stable Diffusion), LLMs, GANs.
  • Reinforcement Learning ‚Äî Policy gradients, Q-learning, Actor-Critic, PPO for sequential decision-making.

E. Evaluation & Metrics

Performance Measurement
  • Classification ‚Äî Accuracy, precision, recall, F1-score, ROC-AUC, confusion matrix.
  • Object Detection ‚Äî mAP (mean Average Precision), IoU (Intersection over Union).
  • Generation Quality ‚Äî BLEU, ROUGE, perplexity, FID, Inception Score, CLIP score.
  • Regression ‚Äî RMSE, MAE, R¬≤, MAPE.
  • Robustness ‚Äî Adversarial testing, out-of-distribution (OOD) detection, calibration metrics.

F. Interview-Ready Topics

Must-know Concepts
  • Forward & Backward Pass ‚Äî Draw a simple network, show how gradients flow backward through layers.
  • Vanishing/Exploding Gradients ‚Äî Causes: deep networks, poor initialization. Solutions: ReLU, skip connections, careful init.
  • Overfitting vs Underfitting ‚Äî Bias-variance tradeoff. Use validation curves to diagnose.
  • Bias & Fairness ‚Äî Dataset bias, label noise, fairness metrics (demographic parity, equalized odds).
  • Scaling Laws ‚Äî Compute, data, and model size tradeoffs. When to scale vs optimize architecture.
  • Reproducibility ‚Äî Set random seeds, use deterministic operations, log hyperparameters, checkpoint models.
  • Debugging Strategies ‚Äî Overfit single batch, check gradients, visualize activations, monitor loss curves.
